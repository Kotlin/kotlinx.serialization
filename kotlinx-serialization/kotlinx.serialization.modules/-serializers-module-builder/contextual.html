contextual

(common, jvm, native) fun <T : Any> contextual(kClass: KClass<T>, serializer: KSerializer<T>): Unit

Adds serializer associated with given kClass for contextual serialization. Throws SerializationException if a module already has serializer associated with a kClass. To overwrite an already registered serializer, SerializersModule.overwriteWith can be used.
