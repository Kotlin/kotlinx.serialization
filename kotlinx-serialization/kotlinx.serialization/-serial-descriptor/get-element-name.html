getElementName

(common, jvm, native) abstract fun getElementName(index: Int): String

Returns a positional name of the child at the given index. Positional name represents a corresponding property name in the class, associated with the current descriptor.

Exceptions

IndexOutOfBoundsException - for an illegal index values.

IllegalStateException - if the current descriptor does not support children elements (e.g. is a primitive)
