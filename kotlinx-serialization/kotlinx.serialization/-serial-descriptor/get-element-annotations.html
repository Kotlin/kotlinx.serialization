getElementAnnotations

(common, jvm, native) abstract fun getElementAnnotations(index: Int): List<Annotation>

Returns serial annotations of the child element at the given index. This method differs from getElementDescriptor(index).annotations by reporting only declaration-specific annotations:

@Serializable
@SomeSerialAnnotation
class Nested(...)

@Serializable
class Outer(@AnotherSerialAnnotation val nested: Nested)

outerDescriptor.getElementAnnotations(0) // Returns [@SomeSerialAnnotation]
outerDescriptor.getElementDescriptor(0).annotations // Returns [@AnotherSerialAnnotation]

Only annotations marked with SerialInfo are added to the resulting list.

Exceptions

IndexOutOfBoundsException - for an illegal index values.

IllegalStateException - if the current descriptor does not support children elements (e.g. is a primitive).
